# 📝 Commit Message Template
# --------------------------------------------------
# <type>(<scope>): <短い変更内容>
# 
# [詳細説明]
# [関連Issueやチケット番号]
# --------------------------------------------------
# type 一覧:
#   feat     : 新機能の追加
#   fix      : バグ修正
#   refactor : コードのリファクタリング
#   style    : コードスタイル修正（動作に影響なし）
#   docs     : ドキュメント修正
#   test     : テスト関連の追加・修正
#   chore    : ビルド・設定・依存パッケージ変更
#   perf     : パフォーマンス改善
#   ci       : CI/CD関連修正
#   hotfix   : 緊急修正
# --------------------------------------------------
# scope例:
#   api, frontend, flutter, infra, db, auth, ci, etc.
# --------------------------------------------------

# feat(api): 予約作成APIのバリデーション追加
